# zhweb-i-journal

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件的形式组织，不针对任何特定域名或网站，仅作为页面资料的集中存放与展示用途。

## 目录结构

```
zhweb-i-journal/
├── pages/          # 存放各个独立 HTML 页面
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子目录或独立文件对应一个归档页面，页面之间相互独立。
- `assets/`：存放页面可能共享的静态资源，供页面引用。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开查看。
- 页面内容可能包含文字、样式、脚本等，但无外部依赖或追踪代码。
- 归档内容不涉及任何特定域名、网站或商业推广，仅供资料保存与展示。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面内容。
- 若需提交页面或资源，请遵循现有目录结构，并确保文件无外部敏感链接。
- 欢迎通过 Issue 或 Pull Request 提出建议或修正，但请勿添加与归档无关的内容。
